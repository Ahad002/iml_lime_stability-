# Data Directory

The SST-2 dataset is **automatically downloaded** from Hugging Face.

**No manual download needed!**

## Dataset Information

- **Name:** Stanford Sentiment Treebank v2 (SST-2)
- **Source:** GLUE Benchmark  
- **Task:** Binary sentiment classification
- **Train:** 67,349 samples
- **Test:** 872 samples
- **Labels:** 0 (negative), 1 (positive)

## Cache Location

Dataset is cached in:
- Mac: `~/.cache/huggingface/datasets/`
